# Bridge - Escola de Idiomas 🌍

Este é o portal web da **Bridge**, uma escola de idiomas focada em conectar alunos ao mundo através do aprendizado de línguas como Inglês, Espanhol, Russo, Chinês e Francês.

O projeto consiste em um site responsivo, com design moderno, animações fluidas e uma área dedicada ao acompanhamento do aluno.

## 🚀 Funcionalidades

- **Página Inicial Dinâmica:** Apresenta um carrossel de imagens interativo e informações sobre os cursos oferecidos.
- **Área do Aluno:** Dashboard personalizado com barra de progresso, cards de recursos de estudo e navegação lateral.
- **Design Responsivo:** Adaptado para dispositivos móveis com menu hambúrguer funcional.
- **Agendamento:** Link direto para agendamento de aulas experimentais.
- **Identidade Visual:** Uso de ícones (Bootstrap Icons) e animações (Animate.css) para uma melhor experiência do usuário.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização avançada utilizando Flexbox, Grid Layout e variáveis para padronização de cores.
- **JavaScript (Vanilla):** Lógica para o carrossel de imagens e controle do menu responsivo.
- **Bootstrap Icons:** Biblioteca de ícones vetoriais.
- **Animate.css:** Biblioteca para animações de entrada e transição.

## 📂 Estrutura do Projeto

```text
site-escola-de-idiomas-main/
├── index.html                 # Página principal
├── README.md                  # Documentação do projeto
└── src/
    ├── assets/
    │   ├── images/            # Logos, banners e fotos
```

## 🔧 Como Executar

1. Clone ou baixe este repositório.
2. Navegue até a pasta raiz do projeto.
3. Abra o arquivo `index.html` em qualquer navegador web moderno.

## 🎨 Destaques de Código

- **Dashboard do Aluno:** Implementado com `display: grid` para separar a sidebar do conteúdo principal de forma eficiente.
- **Responsividade:** Uso de `@media queries` para ocultar elementos complexos em telas menores e ajustar o layout da navegação.
